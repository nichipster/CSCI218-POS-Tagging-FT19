# CSCI218 Group Project - Part-of-Speech Tagging

**Course:** CSCI218 Foundations of AI  
**Topic:** Natural Language Processing - Part-of-Speech Tagging  
**Group:** FT19

## Project Overview

This project implements and evaluates various approaches to Part-of-Speech (POS) tagging - the task of assigning grammatical categories (noun, verb, adjective, etc.) to words in text.
